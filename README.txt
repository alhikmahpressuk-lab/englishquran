# The English Qur'an website

This folder contains a static website prepared for GitHub Pages.

## Files
- `index.html`
- `styles.css`
- `app.js`
- `data/translation.json`
- `CNAME`
- `.nojekyll`

## What is included
- All 114 surahs
- Exact English manuscript text extracted from the uploaded PDF
- Arabic text loaded in the browser from the Qur'an API
- Translator name and edition information on the website

## GitHub upload
1. Create a public repository
2. Upload all files from this folder
3. In GitHub, open Settings, then Pages
4. Set Source to `Deploy from a branch`
5. Choose `main` and `/(root)`
6. Save

## Custom domain
This pack includes a `CNAME` file already set to `englishquran.uk`.

## Note
If you later want the Arabic text embedded directly into the website files rather than loaded from the API, that can be added in a later version.
